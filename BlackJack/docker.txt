# docker.txt

Instrukcja uruchomienia bazy MongoDB dla projektu Blackjack Node.js

1. Uruchomienie kontenera MongoDB:

docker run -d -p 27017:27017 --name mongo mongo

Parametry:
- `-d` – uruchomienie w tle
- `-p 27017:27017` – mapowanie portu MongoDB na lokalny komputer
- `--name mongo` – nazwa kontenera
- `mongo` – oficjalny obraz MongoDB

2. Sprawdzenie działającego kontenera:

docker ps

Powinieneś zobaczyć kontener `mongo` uruchomiony i nasłuchujący na porcie 27017.

3. Podłączenie Node.js do MongoDB:
- W pliku `src/data/connection.js` ustaw URI MongoDB:
  ```
  const { MongoClient } = require('mongodb');
  const client = new MongoClient('mongodb://localhost:27017');
  const dbName = 'blackjack';
  ```
- Teraz projekt Node.js będzie mógł łączyć się z bazą MongoDB.

4. Zatrzymanie kontenera:

docker stop mongo


5. Usunięcie kontenera:

docker rm mongo
